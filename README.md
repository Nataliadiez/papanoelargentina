# 🎅 Papá Noel Argentina — Sitio Institucional

Sitio web de **CG Producciones**, dedicado a la producción de experiencias navideñas en shoppings y eventos.

El objetivo es presentar los servicios, mostrar la trayectoria y facilitar el contacto con potenciales clientes.

## ✨ Sobre el proyecto

- SPA desarrollada en Angular
- Navegación por secciones (scroll)
- Diseño visual enfocado en la experiencia navideña
- Componentes modulares (Hero, Nosotros, Shoppings, Empresas, Contacto)

---

## 🚀 Stack

- Angular
- TypeScript
- Bootstrap 5
- CSS puro
